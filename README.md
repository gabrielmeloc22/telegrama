# Telegrama

A telegram clone using Graphql

## Tech Stack

- Next.js, Relay, Tailwind
- GraphQL, Node.js
- MongoDB, Redis (pubSub)
- JWT

## Features

- Real-time messaging
- User authentication and authorization
- Group chats
- Typing indicators
